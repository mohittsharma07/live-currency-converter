# 💱 Currency Converter App

A simple and responsive currency converter web application built using HTML, CSS, and JavaScript. It provides real-time currency exchange rates along with country flags for better user experience.

---

## 🚀 Features
- Convert currencies in real-time
- Country flags for selected currencies
- Swap currency option
- Simple and responsive UI

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Currency API

---

## 📌 How It Works
1. Enter amount  
2. Select currencies (From / To)  
3. Click "Get Exchange Rate"  
4. See converted value instantly
